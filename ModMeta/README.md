# Chen's Classic Items

## Description

An extension mod for **ThinkInvis.ClassicItems**!

The goal is the same: Add items from Risk of Rain 1 that did not make it to Risk of Rain 2.

## Installation

Use **r2modman** mod manager to install this mod.

### Current Additions
#### Tier 1
- Coming soon!
#### Tier 2
- Panic Mines
#### Tier 3
- AtG Missile Mk. 2
